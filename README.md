# Aptitude Study Platform

A modern, clean, and user-friendly web application designed to help students prepare for competitive exams efficiently.

🔗 **Live Demo:**  
👉 https://aptitudeace.netlify.app/

---

## Overview

This platform helps students organize and track their preparation in:

- 📊 Quantitative Aptitude  
- 🧠 Logical Reasoning  
- 📖 Verbal Ability  

The goal was to build a distraction-free study environment with simple progress tracking and an intuitive interface.

---

## Features

- Categorized aptitude topics
- Mark / Unmark topics as completed
- Dynamic progress tracking per category
- Progress saved using LocalStorage
- Search functionality within topics
- Light / Dark mode toggle
- Fully responsive design
- Clean, modern UI (No frameworks used)

---

## Tech Stack

- HTML5
- CSS3 (Custom UI Design)
- Vanilla JavaScript
- Browser LocalStorage API

No external UI frameworks or libraries were used.

---

## How It Works

- Topics are dynamically loaded based on selected category.
- Users can mark or unmark topics as completed.
- Progress bar updates instantly.
- Completion data is stored in LocalStorage.
- Progress persists even after refreshing the page.
- Dark mode preference can be toggled anytime.
